BigText Makes Text Big
============================

Examples
----------------------------

### Normal Usage

    <div id="bigtext">
        <div>BIGTEXT</div>
        <div>Makes Text Big</div>
    </div>
    <script>
    $('#bigtext').bigtext();
    </script>

### Using a List (ordered/unordered)

    <ol id="bigtext">
        <li>BIGTEXT</li>
        <li>Makes Text Big</li>
    </li>
    <script>
    $('#bigtext').bigtext();
    </script>
